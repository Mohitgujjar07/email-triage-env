# 📩 Email Triage AI

An intelligent system that classifies emails into actions:

* Reply
* Archive
* Escalate
* Snooze

## 🚀 Features

* FastAPI backend
* Rule-based email classification
* Web UI for testing
* Docker support

## 🛠️ Tech Stack

* Python
* FastAPI
* Docker

## ▶️ Run Locally

pip install -r requirements.txt
python app.py

## 🐳 Run with Docker

docker build -t email-triage
docker run -p 7860:7860 email-triage

## 🎯 Use Case

Automates email management and improves productivity.
